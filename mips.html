MIPS Simulator

Project Background

I developed a 32-bit MIPS, 3 staged pipelined microprocessor for the final project of my computer system architecture course. I implemented the MIPS instruction set including R, J, I type and Load Store instructions. And, designed a verification logic which reads instructions and data from a file and simulates the results of each cycle.
