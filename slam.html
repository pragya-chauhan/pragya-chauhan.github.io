Project Background

My interest in robotics took me to an interesting class on Autonomous Mobile Robots. The final project involved mapping a library dataset using Simultaneous Localization and Mapping (SLAM) in visualization tool RVIZ with Robot Operating System (ROS). I obtained a grid map from laser range data implementing GMapping SLAM algorithm.
